# MinesweeperJCompose

This is a Minesweeper game written in Kotlin using Jetpack Compose.

<img src="Screenshot1.png" height="500" /> <img src="Screenshot2.png" height="500" />


Some cool features:
- MVVM architecture
- Material3 Dynamic Color
- Predictive back gesture
